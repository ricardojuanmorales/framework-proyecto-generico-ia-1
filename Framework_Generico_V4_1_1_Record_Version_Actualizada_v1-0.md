## 1. Declaración de versión

El **Framework Genérico V4.1.1** queda definido como una **actualización menor operativa** del Framework Genérico V4.1.0-estable.

V4.1.1 no reescribe V4.1.0.  
V4.1.1 no sustituye V4.0.0 como base histórica.  
V4.1.1 convierte la arquitectura aprobada en una estructura navegable, organizada y utilizable dentro de un repositorio.

Su función principal es habilitar el manejo operativo del proyecto mediante:

```text
cartapacios estructurados;
documentos colocados según función;
placeholders nada.md para GitHub;
README principal;
manifest de repositorio;
árbol documental;
paquete ZIP listo para subir a GitHub;
continuidad para evolución futura.
```
