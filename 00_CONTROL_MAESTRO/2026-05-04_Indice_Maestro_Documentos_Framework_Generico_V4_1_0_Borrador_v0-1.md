## 0. Declaración de propósito

Este índice maestro organiza los documentos activos, base, derivados, de gobernanza, de continuidad y de archivo relacionados con la apertura del **Framework Genérico V4.1.0-borrador**.

Su función es permitir que una persona, equipo, asistente IA o sistema documental pueda responder rápidamente:

```text
¿Qué documentos existen?
¿Cuáles son base y cuáles son derivados?
¿Qué documentos gobiernan decisiones?
¿Qué documentos activan perfiles?
¿Qué documentos deben sincronizarse con wiki?
¿Qué documentos pertenecen al archivo histórico?
¿Qué falta para estabilizar V4.1.0?
```

Este índice no reescribe documentos. Los ubica, clasifica y conecta.

## 11. Reglas para actualizar este índice

Toda actualización debe indicar:

```text
Fecha:
Documento añadido:
Tipo:
Versión:
Estado:
Ubicación lógica:
Documento base relacionado:
Decisión relacionada:
Riesgo:
Impacto:
Recomendación de continuidad:
```

No se debe añadir un documento nuevo al índice si:

1. no tiene propósito claro;
2. duplica otro documento;
3. no tiene versión;
4. no tiene estado;
5. no se sabe qué cartapacio lo contiene;
6. no se sabe si es base, derivado, gobernanza, wiki o archivo.

## 12. Decisión sugerida

```text
D-A29 — Crear el Índice Maestro de Documentos del Framework Genérico V4.1.0-borrador como documento de control, navegación, trazabilidad y continuidad para ordenar documentos base, derivados, gobernanza, wiki, archivo y próximos productos sin alterar la macroarquitectura V4.0.0.
```

**Estado sugerido:** Aceptada para continuidad documental.  
**Tipo de cambio:** Ajuste menor de gobernanza documental.  
**Justificación:** Evita dispersión, duplicación y pérdida de trazabilidad durante la apertura V4.1.0.  
**Impacto:** Alto para navegación, continuidad, archivo y operación humano-IA.  
**Riesgo:** Bajo si se mantiene como índice, no como reescritura.  
**Documentos afectados:** README V4.1.0, changelog, registro de decisiones, primer, manifest futuro.  
**Versión afectada:** V4.1.0-borrador v0.1.  
**Recomendación:** Registrar D-A29 en la próxima actualización del Registro de Decisiones.
