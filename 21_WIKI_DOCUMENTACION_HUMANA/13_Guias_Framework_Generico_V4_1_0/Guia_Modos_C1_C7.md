# Guía Caleidoscopio C1-C7

## Qué es Caleidoscopio

Caleidoscopio es la lógica integradora del Framework Genérico V4.1.0. No es un cuarto perfil tradicional. Orquesta Programador Humanista, Investigador Transdisciplinario y Artista Transdisciplinario para producir emergencia transdisciplinaria.

## Regla central

No basta sumar perfiles. Debe emerger algo que no podía producirse desde un solo perfil.

## Modos

| Modo | Combinación | Uso |
|---|---|---|
| C1 | Programador + Investigador | Apps, dashboards, repositorios, pipelines |
| C2 | Investigador + Artista | Investigación-creación, memoria, publicación multimodal |
| C3 | Artista + Programador | Galería interactiva, avatar, instalación digital |
| C4 | Tres perfiles | AVAT, REA multimodal, ecosistema agéntico |
| C5 | Pedagógico | Curso, módulo, OVAT, ruta formativa |
| C6 | Comunitario | Archivo vivo, mapa interactivo, memoria territorial |
| C7 | Institucional | Laboratorio, programa, manual, wiki institucional |

## Cuándo activar Caleidoscopio

Activar cuando el problema requiere integración entre investigación, creación, programación, documentación, ética, accesibilidad y transferencia.

## Riesgos

- Mezclar sin método.
- Llamar integración a una suma de tareas.
- Publicar sin derechos, licencias o accesibilidad.
- Programar sin contexto.
- Crear sin documentación.
- Investigar sin transferencia.

## Evidencia mínima

- Problema complejo.
- Perfiles activados.
- Modo C1-C7.
- Workflow integrado.
- Producto emergente.
- Evaluación multidimensional.
- Documentación y archivo.
