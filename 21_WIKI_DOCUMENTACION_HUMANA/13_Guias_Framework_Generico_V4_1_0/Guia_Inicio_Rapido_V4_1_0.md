# Guía de Inicio Rápido V4.1.0

## Objetivo

Permitir que una persona o asistente IA active el Framework Genérico V4.1.0 sin reconstruir todo el contexto.

## Ruta mínima

```text
1. Definir propósito.
2. Elegir nivel N1-N4.
3. Activar perfil o Caleidoscopio.
4. Seleccionar documento base.
5. Aplicar Gate correspondiente.
6. Aplicar CHECKS.
7. Producir evidencia.
8. Actualizar documentación.
9. Revisar WIKI_SYNC.
10. Cerrar y archivar.
```

## Documentos que se abren primero

```text
README Framework Genérico V4.1.0
Índice Maestro
Manifest JSON
Matriz N1-N4
Matriz Gates
CHECKS
WIKI_SYNC
Primer de Continuidad
```

## Selección rápida

| Necesidad | Activar |
|---|---|
| Código, app, repositorio, automatización | Programador Humanista |
| Investigación, fuentes, evidencia, bibliometría | Investigador Transdisciplinario |
| Obra, avatar, memoria, curaduría | Artista Transdisciplinario |
| Integración entre perfiles | Caleidoscopio |
| Transferencia institucional | N4 / C7 |

## Regla de oro

No producir documentos nuevos sin propósito, versión, estado, evidencia y próximo paso.
