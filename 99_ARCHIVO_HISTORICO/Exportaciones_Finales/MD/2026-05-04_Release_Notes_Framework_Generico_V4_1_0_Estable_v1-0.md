## 1. Resumen

Framework Genérico V4.1.0-estable es una evolución controlada de V4.0.0. Integra formalmente la capa de **Perfiles Transversales** y **Caleidoscopio** junto con una infraestructura de gobernanza documental, validación, wiki, archivo y continuidad.

Esta versión no borra V4.0.0. La conserva como base histórica y añade una capa operativa para proyectos complejos humano-IA.

## 7. Cómo comenzar

```text
1. Abrir README V4.1.0.
2. Revisar Índice Maestro.
3. Identificar nivel N1-N4.
4. Seleccionar perfil o modo Caleidoscopio.
5. Aplicar Matriz Gates.
6. Aplicar CHECKS.
7. Revisar WIKI_SYNC.
8. Documentar avance.
9. Cerrar con continuidad y archivo.
```

---

## 9. Riesgos conocidos

| Riesgo | Estado | Mitigación |
|---|---|---|
| Wiki secundaria incompleta | No bloqueante | Completar en ciclo posterior |
| Automatización prematura | Controlado | Mantener como pendiente N4 |
| Uso sin aplicar CHECKS | Medio | Reforzar guía de inicio |
| Caleidoscopio usado como suma superficial | Medio | Usar matriz C1-C7 y Gates |
| Perfiles v0.1 requieren refinamiento | Bajo | Preparar v0.2 posterior |
