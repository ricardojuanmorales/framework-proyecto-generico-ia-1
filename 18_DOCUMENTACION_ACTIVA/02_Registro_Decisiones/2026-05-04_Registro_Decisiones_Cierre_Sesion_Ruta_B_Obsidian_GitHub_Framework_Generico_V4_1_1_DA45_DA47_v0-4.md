## 4. D-A47 — Ruta B híbrida Obsidian-GitHub

**Decisión:**  
Implementar la Ruta B híbrida para que la raíz del repositorio completo funcione como bóveda Obsidian, preservando GitHub como fuente oficial y usando Obsidian como segundo cerebro liviano.

**Estado:** Ejecutada.  
**Tipo:** Ajuste menor operativo.  
**Justificación:**  
Permite que la Wiki humana, la documentación activa, el archivo histórico y los documentos oficiales sean navegables desde Obsidian sin duplicarlos.

**Impacto:**  
Alto para operación diaria, sincronización, navegación, seguimiento y preservación.

**Riesgo:**  
Bajo si se mantiene la regla de sincronización:

```text
GitHub = fuente oficial.
Obsidian = navegación e interpretación.
ChatGPT = orquestación.
Issues = seguimiento.
```
