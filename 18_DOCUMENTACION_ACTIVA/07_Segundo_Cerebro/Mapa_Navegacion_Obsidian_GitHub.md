# 🗺️ Mapa de Navegación Obsidian-GitHub

**Ubicación:** `18_DOCUMENTACION_ACTIVA/07_Segundo_Cerebro/Mapa_Navegacion_Obsidian_GitHub.md`

---

## Arquitectura de memoria

```text
ChatGPT / sesión
→ produce, evalúa, sintetiza y cierra

GitHub / repositorio
→ conserva verdad oficial, versiones, releases e historial

Obsidian / segundo cerebro
→ facilita navegación diaria, interpretación y monitoreo

GitHub Issues
→ convierte próximos pasos en tareas trazables
```

---

## Flujo recomendado

```text
git pull
→ abrir repo como vault Obsidian
→ abrir Home Obsidian
→ revisar Dashboard e Issues vivos
→ trabajar sesión
→ actualizar documentos y dashboard
→ git add / commit / push
```

---

## Reglas

```text
No duplicar documentos oficiales completos en Obsidian.
No tomar decisiones solo en Obsidian.
No cerrar issues sin evidencia.
No producir documentos sin cartapacio destino.
No modificar V4.0.0 ni V4.1.0-estable sin decisión explícita.
```
