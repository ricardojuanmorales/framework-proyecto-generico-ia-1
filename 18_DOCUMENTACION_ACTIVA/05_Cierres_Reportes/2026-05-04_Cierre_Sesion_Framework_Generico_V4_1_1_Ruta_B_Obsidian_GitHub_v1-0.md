## 1. Resumen ejecutivo

Se cierra la sesión en la que el Framework Genérico V4.1.1 pasó de repositorio modelo a sistema operativo de continuidad conectado con GitHub y Obsidian.

La decisión central de la sesión fue adoptar e implementar una **Ruta B híbrida**:

```text
Repositorio completo = bóveda Obsidian
GitHub = fuente oficial versionada
Obsidian = segundo cerebro liviano y navegable
Wiki humana = accesible desde Obsidian
Issues ligeros = seguimiento operativo
ChatGPT = orquestador humano-IA de producción, evaluación y cierre
```

La sesión deja preparado el proyecto para trabajar diariamente desde Obsidian sin perder la estructura oficial del repositorio GitHub.

## 3. Decisión operativa principal

```text
D-A47 — Implementar Ruta B híbrida Obsidian-GitHub para que la raíz completa del repositorio funcione como bóveda Obsidian, preservando GitHub como fuente oficial, Obsidian como segundo cerebro liviano, la Wiki humana como navegación interna y los Issues ligeros como mecanismo de seguimiento.
```

**Estado:** Aceptada por instrucción humana y ejecutada.  
**Tipo de cambio:** Ajuste menor operativo de continuidad y flujo de trabajo.  
**Impacto:** Alto para navegación, seguimiento, preservación, uso diario y coherencia del proyecto.  
**Riesgo:** Bajo si se mantiene la regla de que GitHub conserva la verdad oficial y Obsidian funciona como interfaz de navegación.
